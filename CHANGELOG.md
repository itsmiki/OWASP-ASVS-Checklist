## [ASVS-5.0.0-v1.0] - 2026-02-28

### Added
- Initial release of ASVS 5.0.0 Assessment Spreadsheet
- **STATISTICS** sheet:
  - Overview of assessment progress
  - Level selection for ASVS assessment (L1/L2/L3)
  - Checkbox controls to include/exclude requirements based on level
  - Automatic calculation of completed vs total requirements
  - Visual chart showing compliance percentage
- Individual sheets for each ASVS category
- Conditional formatting to strike through requirements that are not being assessed
- Automatic exclusion of unchecked requirements from statistics
- **About / Metadata** sheet:
  - Tool name, version, author, release date
  - Short description of the tool
